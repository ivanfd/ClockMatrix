Завантажувач ПО для PIC18F4525 по USART(TinyBootloader). Швидкість 9600.
Адаптовано під годинник на матричних індикаторах. Програмування через
BlueTooth.

Для завантаження ПО в годинник, відкриваємо "tinybldWin.exe",
вибираємо файл ПО "xxx.HEX", переходимо на вкладку "Terminal", 
шлемо команду $z(скидання контролера),
закриваємо порт і натискаємо Write Flash.
